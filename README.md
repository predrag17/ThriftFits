# ThriftFits
### *Overview:*

*ThriftFits is your premier platform for buying and selling secondhand clothing. Our app connects fashion enthusiasts looking to refresh their wardrobe with sellers eager to find new homes for their gently loved pieces. With user-friendly features and advanced search options, ThriftFits makes it simple to discover unique finds and reduce fashion waste.*

## Types of users:

> 1. User - this is either a buyer or seller.
> 2. Admin - this user manages the app if someone post inappropriate ad or inappropriate comment etc.
